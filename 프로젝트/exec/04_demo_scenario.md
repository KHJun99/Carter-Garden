# 4) 시연 시나리오 (화면/클릭 위치 포함)

## 사전 준비
1. Backend 실행 (`server/backend/run.py`, 포트 5001)
2. Frontend 실행 (`raspberry_pi/frontend`, Vite 5173)
3. Jetson ROS/카메라/rosbridge 연결 확인
4. RFID 서버(8765) 송신 가능 상태 확인

## 시연 순서

### 1. 홈 화면 (`/`)
- 화면: 회원 / 비회원 선택 카드 2개
- 클릭: 오른쪽 `비회원` 카드 클릭
- 결과: 사용자 등록 화면(`/register-yolo`) 이동

### 2. 사용자 등록 화면 (`/register-yolo`)
- 화면: 카메라 라이브 피드 + 하단 단일 버튼
- 클릭: 하단 `등록 완료 및 쇼핑 시작` 버튼
- 조건: ROS status `REGISTER_DONE` 수신 후 버튼 활성화
- 결과: 상품 목록(`/products`) 이동

### 3. 상품 목록 (`/products`)
- 화면 구성
  - 좌측: 카테고리
  - 중앙: 상품 카드 리스트
  - 우측: AI 추천 상품
  - 하단: `따라가기 모드`, `쇼핑 종료`
- 클릭 A: 중앙 상품 카드 우측 카트아이콘(상품 담기/이동)
- 결과 A: `/guide`로 이동, 목적지까지 주행 안내
- 클릭 B: 우상단 카트 아이콘
- 결과 B: 장바구니(`/cart`) 이동

### 4. 이동 안내 (`/guide`)
- 화면: 좌측 상태패널 + 우측 맵
- 클릭: 하단 좌측 `목적지 변경` 또는 우측 `일시 정지/주행 시작`
- 결과: 상품목록 복귀 또는 주행 제어

### 5. 장바구니 (`/cart`)
- 화면: 상품 목록 + 결제 요약
- 클릭 A: 각 상품의 휴지통 아이콘
- 동작 A: 삭제 다이얼로그 표시 후 RFID 재태깅 시 수량 감소/삭제
- 클릭 B: 우측 결제 버튼(`결제하기`)
- 결과 B: 결제 화면(`/payment`) 이동

### 6. 결제 (`/payment`)
- 화면: 쿠폰 선택 + 결제수단(카드/간편결제/포인트)
- 클릭 순서
  1) 쿠폰 카드 선택(선택 사항)
  2) 결제수단 카드 클릭
  3) 하단 우측 `결제 및 쇼핑 종료` 클릭
  4) 모달에서 `결제하기` 클릭
- 결과: `/finish` 이동

### 7. 쇼핑 종료 (`/finish`)
- 선택지 2개
  - 왼쪽 `여기서 바로 반납`
  - 오른쪽 `주차장까지 이동`
- 클릭 A: `여기서 바로 반납`
- 결과 A: `/thanks` 이동
- 클릭 B: `주차장까지 이동`
  - 모달 입력창에 차량번호(4자리 이상) 입력
  - 검색결과 항목 클릭 후 `이동 시작`
- 결과 B: `/guide?from=finish...` 이동

### 8. 감사 화면 (`/thanks`)
- 화면: 반납/종료 안내 문구
- 동작: ARRIVED 수신 후 2초 뒤 홈(`/`) 자동 복귀

## 시연 포인트
- RFID 태깅으로 장바구니 실시간 반영
- ROS Bridge 연동 실시간 상태/위치 갱신
- 추천 상품 및 결제 플로우 연계
