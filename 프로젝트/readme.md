# 스마트 자율주행 쇼핑 카트 PoC

> **LiDAR SLAM 기반 자율주행과 센서 융합을 통한 미래형 쇼핑 자동화 솔루션**

## 1. 프로젝트 개요

* **진행 기간**: 2026.01.12 ~ 2026.02.08 (4주)
* **목표**: 축소 마트 환경에서 LiDAR SLAM 기반 자율주행과 RFID 장바구니 시스템이 결합된 지능형 카트 시스템 구현
* **핵심 가치**: 쇼핑의 번거로움(카트 이동, 상품 찾기)을 해결하고 리테일 운영 효율을 극대화하는 자율주행 기술 실증

---

## 2. 주요 기능

1. **상품 위치 자율주행**: 웹 UI에서 상품 선택 시 진열대 좌표로 최단 경로 주행
2. **실시간 장바구니 관리**: RFID 태그 인식을 통한 실시간 물품 추가 및 삭제(재태깅 로직)
3. **사용자 추종 모드**: 카메라 비전을 활용한 일정한 거리의 사용자 추적 주행
4. **자동 반납 및 안내**: 결제 완료 후 출구 가이드 주행 및 대기 장소 복귀

---

## 3. 시스템 아키텍처

본 프로젝트는 **분산 컴퓨팅 구조**를 채택하여 연산 효율과 시스템 안정성을 확보

* **Jetson Orin Nano**: 고성능 AI 연산 및 로봇 주행 제어(ROS, SLAM, Navigation) 담당
* **Raspberry Pi 5**: 센서 데이터(RFID) 처리 및 Flask 웹 서버 운영 담당
* **Communication**: ROS-Web Bridge와 Socket.io를 활용한 실시간 양방향 데이터 통신
